# Weather App

## 🌦️ Project Overview
This is a simple Weather App that allows users to check the current weather of any city. It uses the Open-Meteo API to fetch real-time weather data.

## 🚀 Features
- Search weather by city name
- Displays temperature and wind speed
- Error handling for invalid cities
- Multi-city support

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Open-Meteo API

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter a city name and click "Get Weather"
